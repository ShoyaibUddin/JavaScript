# DOM Document Object Model

Dom is a Document Object Model & Programming interface for HTML that defines HTML objects, properties, methods & events to get, change, add, or delete HTML elements

একটি প্রোগ্রামিং ইন্টারফেস এইচটিএমএল এবং এক্সএমএল নথি DOM একটি ওয়েব পৃষ্ঠার কাঠামোকে একটি গাছের মতো কাঠামো হিসাবে উপস্থাপন করে, যেখানে গাছের প্রতিটি নোড পৃষ্ঠায় একটি ভিন্ন উপাদান বা বস্তুর প্রতিনিধিত্ব করে, যেমন শিরোনাম, অনুচ্ছেদ, চিত্র এবং ফর্ম।

- change HTML Element
- change HTML attributes
- can change CSS style
- add HTML elements and attributes
- remove HTML elements and attributes
- react to HTML events

> P O E M

> P = Property
> O = Object
> E = Event
> M = Method

Method মানে হলো Function
Ex

```
getElementById();
```

> Method -> HTML element

> Property - > HTML Valus

Object এর ভিতরে Method এবং Property থাকে।

```
document.getElementById("example").innerHTML = "Hello World";
```

উপরের Example এ Document হলে একটি Object

.getElementById() একটি মেথট বা ফাংশন

.innerHTML হলে একটি Property
